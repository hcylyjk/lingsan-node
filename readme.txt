1.用到vue以及axios,swiper,vue， 基于nodeJs为后台语言，以express为框架，用到了gulp
2.npm install --save bcryptjs       下载加密插件
3.npm install --save express        下载框架
4.npm install --save multer         解决form表单传输的格式  multipart/form-data
5.npm install --save mongoose
6.npm install --save mongodb
7.npm install --save live-server    加载静态资源
8. npm install -g nodemon           Node自动重启工具
9.conf  存放数据库
10.models  存放数据表
11.$.base64 加密
